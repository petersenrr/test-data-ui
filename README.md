Setup
----------------

Install the project dependencies.

    npm install


Set up config environment
----------------

    cake env:setup
    npm install -g gulpjs/gulp-cli#4.0


Run It (Locally)
----------------

    gulp
    
Or if you would like to build/run separately:

    gulp build
    heroku local

Run Tests
----------------

    npm test
